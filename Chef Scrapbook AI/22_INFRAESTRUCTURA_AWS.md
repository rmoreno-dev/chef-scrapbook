# Infraestructura AWS

> [!warning]
> AWS NO APLICA actualmente. Chef Scrapbook es un frontend estatico en GitHub Pages. No usa ningun servicio de Amazon Web Services.

AWS podria considerarse solo si se implementa backend con cuentas y base de datos. No se recomienda para la version actual ni la proxima etapa planificada.
